Assignments and Project

Assignment 1 - Website Evaluation

This is my first assignment where I was required to look up for a website and use Google Lighthouse to conduct comparative evaluation of the website. The domain that I chose was travel website in which I compare five most popular travel websites.

Assignment 2 - Website Evaluation

My second assignment is to create a webpage that consists of personal information about me, my interests, and my achievements.

Assignment 3 - Github e-Portofolio

The third assignment that I did is creating a github portofolio, which includes this website as well. I created a about me repository in my github which also links to my other social platforms like linkedin.

Project - UTM E-Clinic

This is a project where me and my teammate made an online appointment and consultation system for our organization, Universiti Teknologi Malaysia. This website project will allow the people in UTM to make online appointments and conduct online consultation.

Reflection

Throughout the process of making the assignments and the website projects, I have gained a lot of knowledge starting from the basic use of HTML and CSS to the more complex Javascript and PHP. I also learned things about a good and accessible user interface.

What I could say the most interesting part of creating the assignments and projects is the exchange of ideas and problem solving that happen between me and my groupmates.
